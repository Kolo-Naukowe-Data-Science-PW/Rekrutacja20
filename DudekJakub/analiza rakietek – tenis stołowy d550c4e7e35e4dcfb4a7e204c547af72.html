Po co to wszystko?

Powyższe dane posłużyły mi do podsumowania poszczególnych statystyk dla poszczególnych przedziałów cenowych, zbudowałem modele predykcji oraz przeanalizowałem znaczenie zmiennych.

Od pomysłu do realizacji

  • Pobranie oraz oczyszczenie danych
dane0 = read.csv("TB.csv") #pobranie danych
dane_1 = dane0 %>%  
  filter(!is.na(Price)&Ratings>0&
	Price!=''&!is.na(Overall)&!is.na(Speed)&!is.na(Stiffness)&!is.na(Control))
  dane_1$Price = as.integer(gsub("\\$","",dane_1$Price))
#filtrowanie pustych danych

  • Dodanie nowych zmiennych
daneS = dane_1 %>%   arrange(Price) %>% 
  mutate(PriceInt = case_when(Price <= 30 ~  1,
                              Price <= 50 ~ 2,
                              Price <= 100 ~ 3,
                              Price <= 200 ~ 4,
                              Price <= 300 ~ 5,
                              Price <= 500 ~ 6,
                              TRUE ~ 7)) %>%   group_by(PriceInt) %>% 
  mutate(meanOverall = mean(Overall)) %>% mutate(meanSpeed = mean(Speed)) %>% 
	mutate(meanRatings = mean(Ratings)) %>% mutate(meanControl = mean(Control)) %>%
  mutate(meanStiffness = mean(Stiffness)) %>% ungroup()
#stworzenie wartości średnich dla każdej statystyki

  • Zapisanie danych średnich w nowej tabeli 
meandane = distinct(daneS, PriceInt, meanOverall, meanSpeed, meanStiffness, 
meanControl, meanRatings)

Na tej podstawie stworzyłem kilka wykresów przedstawiających zależności poszczególnych danych względem przedziałów cenowych

Jak widać w większości przypadków średnie wartości statystyk rosły wraz przyrostem ceny.

Inaczej jest w przypadku średniej liczby recenzji. Możemy zatem wnioskować, że przedział najcześciej oceniany był również najchętniej wybierany przez użytkowników.

Zostawiając planowanie zakupów na dalszy plan zacząłem budować modele predykcji poszczególnych statystyk

  • Stworzenie tabeli dla modeli i testów

dane_model_A = daneS %>%  select(Speed, Overall, Stiffness, Control, Price)

names(dane_model_A)[1]="Prędkość"
names(dane_model)[2]="Ogólna"
names(dane_model_A)[3]="Sztywność"
names(dane_model_A)[4]="Kontrola"
names(dane_model_A)[5]="Cena"
#nazwanie danych w tabeli zamiast w poszczególnych wykresach było wygodniejsze
dane_model = dane_model_A %>%  sample_frac(0.8, replace = FALSE)

dane_model_test = anti_join(dane_model_A, dane_model)
#wybranie losowej próbki dla modeli oraz przydzielenie reszty na potrzeby testów

  • Tworzenie modeli
Cset = trainControl(method = "repeatedcv", number = 10, repeats = 1, verboseIter = TRUE)

model_Sp.cv = train(Prędkość~., data=dane_model,
